Política de Privacidade – Forca Bíblica

Última atualização: 24 de maio de 2025

Responsável: Cristais Studios

1. Coleta de Dados

O aplicativo Forca Bíblica pode coletar os seguintes tipos de dados dos usuários:

  • Dados Pessoais: nome, e-mail (caso fornecido voluntariamente pelo usuário).
  • Dados Técnicos: informações de dispositivo, versão do sistema, identificadores anônimos, logs de erro e dados de uso.
  • Dados Sensíveis: não coletamos dados sensíveis como origem racial, convicção religiosa (exceto conteúdo bíblico temático), dados de saúde etc.

Observação: O conteúdo do jogo é de temática cristã, mas não exige declaração de religião ou dados sensíveis.

2. Uso dos Dados

Os dados coletados são utilizados para:

  • Melhorar a experiência do usuário;
  • Corrigir erros e melhorar o desempenho;
  • Enviar comunicações, se autorizado pelo usuário;
  • Cumprir obrigações legais ou regulatórias.

3. Compartilhamento de Dados

Cristais Studios não vende nem compartilha dados pessoais para fins comerciais. Dados podem ser compartilhados com:

  • Serviços de análise (ex: Google Firebase);
  • Serviços de armazenamento em nuvem;
  • Autoridades legais, se exigido por lei.

4. Segurança dos Dados

Adotamos medidas de segurança para proteger os dados contra acesso não autorizado, perda ou tratamento indevido:

  • Criptografia na transmissão dos dados (TLS/SSL);
  • Acesso restrito a pessoal autorizado;
  • Armazenamento protegido contra acesso externo.

5. Retenção e Exclusão de Dados

  • Retenção: Os dados são mantidos enquanto forem necessários para os fins informados nesta política.
  • Exclusão: O usuário pode solicitar a exclusão de seus dados pelo e-mail abaixo. A exclusão será processada em prazo razoável.

6. Direitos dos Usuários

O usuário tem direito a:

  • Acessar os dados armazenados;
  • Corrigir dados incorretos;
  • Solicitar a exclusão dos dados;
  • Revogar consentimentos dados anteriormente.

Contato: contato@cristaisstudios.com.br

Cristais Studios poderá alterar esta Política de Privacidade a qualquer momento, com aviso dentro do app ou por outros meios. Recomendamos revisitar este documento periodicamente.
